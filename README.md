### what i'm actually building

**Agent J** — a two-phone Android agent system. Phone A runs a Python agent loop with ADB, OCR, and multi-AI routing (Gemini → ChatGPT → Claude). Phone B runs a Claude bridge. The whole thing orchestrates real UI automation: opening apps, reading screens, copying and pasting, summarising — no emulator, no desktop, just two physical phones and Termux.

> 30+ sessions in. CoordScaler for device-agnostic coordinates. n8n orchestration layer. Self-heal pipeline. Still not done.

**Checkmate** — productivity/distraction-blocking app for NEET exam students. Accessibility service that monitors app usage, floating attention overlay, adaptive coaching via LLM, and an autonomous screenshot-to-Telegram feature for guardians. Built for the pressure I know personally.

**Nexus** — full Discord clone. Kotlin/Jetpack Compose frontend. Fastify + Prisma + TypeScript backend. Runs entirely on one Android device via Termux. Gateway events, token refresh, real-time channels — the whole thing.
